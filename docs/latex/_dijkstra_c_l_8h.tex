\subsection{Detailed Description}
A simple wrapper class for the Open\+C\+L implementation of the Parallel \hyperlink{namespace_dijkstra}{Dijkstra} Algorithm for all-\/pair or selected-\/pair shortest distances calculation.

This code is adapted from the example code (by Dan Ginsburg) for the book Open\+C\+L Programming Guide (by Munish, et al.) (\href{http://www.heterogeneouscompute.org/?page_id=5}{\tt http\+://www.\+heterogeneouscompute.\+org/?page\+\_\+id=5}) with some tweaks and one data reading synchronization bug fixed for large datasets and adding multiple source shortest distances support. Both of the code and the original code implements the parallel \hyperlink{namespace_dijkstra}{Dijkstra} Algorithm from the paper \char`\"{}\+Accelerating large graph algorithms on the G\+P\+U using C\+U\+D\+A\char`\"{} (by Harish and Narayanan).
